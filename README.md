# coffeetime
회사 카페에서 커피를 주문후에 커피가 나올 때가 되면 알람을 주는 웹/앱

# 구성
- 사용자는 대기표 번호를 입력
- 대기표 번호와 현재 번호가 가까워지면 알람 발생
- 현재 번호 계산은
  -   커피 현황판 웹으로 부터 이미지를 받아와서
  - 숫자를 인식   

# 빌드 및 실행 방법
## 서버 실행
1. 서버 폴더로 이동
    - `cd test_server`
2. 필요 패키지 설치
    - `npm install -dev`
3. 서버 실행
    - `node index.js`

## 프론트 실행
1. 최상위 폴더로 이동
    - `cd ..`
2. 필요 패키지 설치
    - `npm install`
3. 프론트 실행
    - `npm start`
4. 브라우저로 접속 (http://localhost:3000) 하면 프론트 화면이 뜬다.

# OCR 기능 추가
이 프로젝트는 Tesseract.js를 사용하여 이미지에서 숫자를 추출하는 기능을 포함합니다.

## Tesseract.js 설치
프로젝트 루트 디렉토리에서 다음 명령어를 실행하여 Tesseract.js를 설치합니다.
```
npm install tesseract.js
```

## OCR 기능 사용 방법
1. 서버를 실행하면 `/images` 엔드포인트가 이미지를 가져오고 OCR을 수행하여 결과를 반환합니다.
2. 프론트엔드에서 이미지를 가져와서 OCR 결과를 화면에 표시합니다.
